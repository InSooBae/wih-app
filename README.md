# What's in Here

React Native App for What's in Here.

## development environment

- Visual Studio Code / macOS 10.15

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install dependency packages.

```shell
yarn install
```

## Usage

```shell
yarn start
```
